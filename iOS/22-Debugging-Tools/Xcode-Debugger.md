# Xcode Debugger (LLDB)

## Breakpoints

```swift
// Conditional Breakpoint
// condition: user.name == "John"

// Log Message
// po user.name
```

## LLDB Commands

```
po variable         // Print object
p variable          // Print value
expr variable = 10  // Modify value
bt                  // Backtrace
c                   // Continue
```
