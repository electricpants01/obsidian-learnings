# Validación de Modelos y Métricas

## Descripción

La validación correcta y elección de métricas apropiadas son cruciales para evaluar modelos de ML. Previene overfitting, data leakage y asegura que el modelo generalice bien a datos nuevos.

## Conceptos Clave

### 1. **Train/Test Split**
- Holdout validation
- Stratified split
- Time-based split
- Random state

### 2. **Cross-Validation**
- K-fold CV
- Stratified K-fold
- Leave-one-out
- Time series CV

### 3. **Métricas de Clasificación**
- Accuracy
- Precision/Recall
- F1-score
- ROC-AUC
- Confusion matrix

### 4. **Métricas de Regresión**
- MSE/RMSE
- MAE
- R²
- MAPE

### 5. **Overfitting vs Underfitting**
- Bias-variance tradeoff
- Learning curves
- Validation curves
- Regularization

### 6. **Imbalanced Data**
- SMOTE
- Class weights
- Stratified sampling
- Threshold tuning

### 7. **Data Leakage**
- Target leakage
- Train-test contamination
- Temporal leakage
- Prevention strategies
