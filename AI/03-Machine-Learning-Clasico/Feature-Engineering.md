# Feature Engineering - Ingeniería de Características

## Descripción

Feature Engineering es el proceso de transformar datos crudos en features que mejoran el rendimiento de modelos de ML. Es frecuentemente más importante que el algoritmo elegido y requiere conocimiento del dominio.

## Conceptos Clave

### 1. **Feature Scaling**
- StandardScaler
- MinMaxScaler
- RobustScaler
- Normalizer

### 2. **Encoding Categórico**
- One-Hot Encoding
- Label Encoding
- Target Encoding
- Frequency Encoding

### 3. **Feature Creation**
- Polynomial features
- Interactions
- Domain-specific
- Aggregations

### 4. **Feature Selection**
- Filter methods
- Wrapper methods
- Embedded methods
- Recursive elimination

### 5. **Temporal Features**
- Date decomposition
- Lag features
- Rolling statistics
- Seasonality

### 6. **Text Features**
- TF-IDF
- Bag of Words
- N-grams
- Word embeddings

### 7. **Missing Data**
- Imputation strategies
- Indicator variables
- MICE
- KNN imputation
