# Algoritmos de Machine Learning No Supervisado

## Descripción

Los algoritmos no supervisados encuentran patrones en datos sin etiquetas. Incluyen clustering, reducción dimensional y detección de anomalías. Son esenciales para análisis exploratorio y compresión de datos.

## Conceptos Clave

### 1. **K-Means Clustering**
- Centroides
- Elbow method
- Silhouette score
- Mini-batch K-means

### 2. **Clustering Jerárquico**
- Agglomerative
- Divisive
- Dendrogramas
- Linkage methods

### 3. **DBSCAN**
- Density-based
- Eps y min_samples
- Outlier detection
- Arbitrary shapes

### 4. **PCA**
- Reducción dimensional
- Explained variance
- Eigenvalues
- Visualization

### 5. **t-SNE**
- Visualización
- Perplexity
- No lineal
- Preservar estructura local

### 6. **Autoencoders**
- Neural network approach
- Encoder-decoder
- Dimensionality reduction
- Anomaly detection

### 7. **Gaussian Mixture Models**
- Probabilistic clustering
- EM algorithm
- Soft assignments
- BIC/AIC
