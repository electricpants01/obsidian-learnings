# Algoritmos de Machine Learning Supervisado

## Descripción

Los algoritmos supervisados aprenden de datos etiquetados para hacer predicciones. Incluyen clasificación (predecir categorías) y regresión (predecir valores continuos). Son fundamentales para la mayoría de aplicaciones de ML.

## Conceptos Clave

### 1. **Regresión Lineal y Logística**
- Modelos lineales
- Ridge y Lasso
- Regularización
- Interpretabilidad

### 2. **Árboles de Decisión**
- Decision trees
- Criterion (Gini, entropy)
- Pruning
- Visualización

### 3. **Random Forests**
- Ensemble de árboles
- Bagging
- Feature importance
- Out-of-bag score

### 4. **Gradient Boosting**
- XGBoost, LightGBM
- Boosting iterativo
- Learning rate
- Early stopping

### 5. **Support Vector Machines**
- Kernel trick
- C y gamma parameters
- Multi-class classification
- SVR para regresión

### 6. **K-Nearest Neighbors**
- Distance metrics
- K selection
- Weighted voting
- Curse of dimensionality

### 7. **Naive Bayes**
- Teorema de Bayes
- Gaussian NB
- Multinomial NB
- Text classification
