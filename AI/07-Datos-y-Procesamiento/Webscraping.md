# Web Scraping - Extracción de Datos Web

## Descripción

Web scraping extrae datos de sitios web para análisis y ML. Útil cuando no hay APIs disponibles. Requiere respetar robots.txt y términos de servicio.

## Conceptos Clave

### 1. **Herramientas**
- BeautifulSoup
- Scrapy
- Selenium
- Requests

### 2. **Técnicas**
- HTML parsing
- CSS selectors
- XPath
- Dynamic content

### 3. **Buenas Prácticas**
- Rate limiting
- User agents
- robots.txt
- Legal considerations

### 4. **Anti-scraping**
- CAPTCHA
- IP blocking
- Proxies
- Rotating headers
