# Transfer Learning

## Descripción

Transfer Learning permite reutilizar modelos pre-entrenados en nuevas tareas, ahorrando tiempo y recursos. Es especialmente útil cuando los datos son limitados.

## Conceptos Clave

### 1. **Conceptos**
- Pretrained models
- Domain adaptation
- Fine-tuning
- Feature extraction

### 2. **Estrategias**
- Freeze layers
- Learning rates
- Progressive unfreezing
- Discriminative fine-tuning

### 3. **Modelos Populares**
- ResNet
- BERT
- GPT
- ViT
- CLIP

### 4. **Aplicaciones**
- Few-shot learning
- Domain adaptation
- Multi-task learning
- Zero-shot learning
