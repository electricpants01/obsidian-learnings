# RNNs, LSTMs y Transformers

## Descripción

Arquitecturas para datos secuenciales. RNNs procesan secuencias, LSTMs resuelven el problema de gradientes, y Transformers revolucionaron NLP con mecanismos de atención.

## Conceptos Clave

### 1. **RNN Básicas**
- Recurrent connections
- Hidden state
- Vanishing gradients
- BPTT

### 2. **LSTM/GRU**
- Memory cells
- Gates
- Long-term dependencies
- Bidirectional

### 3. **Transformers**
- Self-attention
- Multi-head attention
- Positional encoding
- BERT/GPT

### 4. **Aplicaciones**
- Machine translation
- Text generation
- Time series
- Speech recognition
