# Generative AI - IA Generativa

## Descripción

Generative AI crea contenido nuevo: imágenes, texto, audio, video. Incluye GANs, VAEs, Diffusion Models y modelos de lenguaje como GPT.

## Conceptos Clave

### 1. **Modelos**
- GANs
- VAEs
- Diffusion Models
- LLMs
- Flow-based

### 2. **Aplicaciones Imagen**
- Text-to-image
- Style transfer
- Inpainting
- Super-resolution

### 3. **Aplicaciones Texto**
- Text generation
- Code generation
- Translation
- Summarization

### 4. **Herramientas**
- Stable Diffusion
- Midjourney
- DALL-E
- GPT-4
- Claude
