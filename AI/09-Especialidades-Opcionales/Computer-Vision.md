# Computer Vision - Visión por Computadora

## Descripción

Computer Vision permite a las computadoras interpretar imágenes y videos. Incluye clasificación, detección de objetos, segmentación y reconocimiento facial.

## Conceptos Clave

### 1. **Tareas**
- Classification
- Object detection
- Segmentation
- Tracking

### 2. **Arquitecturas**
- CNNs
- ResNet
- YOLO
- Mask R-CNN
- Vision Transformers

### 3. **Técnicas**
- Data augmentation
- Transfer learning
- Multi-scale
- Attention

### 4. **Aplicaciones**
- Autonomous vehicles
- Medical imaging
- Surveillance
- AR/VR
