# Resolución de Problemas

## Descripción
Resolución de problemas descompone challenges complejos en pasos manejables. Framework: define el problema, explora soluciones, implementa, evalúa resultados. ML context: debugging models, optimizing pipelines, handling edge cases. Approach sistemático, iterativo, data-driven.

## Framework
1. **Define**: ¿Cuál es el problema exacto?
2. **Analiza**: ¿Qué datos/recursos disponibles?
3. **Diseña**: ¿Posibles soluciones? Trade-offs
4. **Implementa**: MVP primero, itera
5. **Valida**: ¿Resuelve el problema? ¿Generaliza?
