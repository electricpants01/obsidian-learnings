# Resolución de Problemas

## Descripción

La resolución de problemas sistemática es esencial para AI Engineers. Incluye entender el problema, diseñar soluciones, implementar y iterar.

## Conceptos Clave

### 1. **Framework**
- Define problem
- Analyze requirements
- Design solution
- Implement
- Evaluate

### 2. **Estrategias**
- Divide and conquer
- Pattern matching
- Simplification
- Analogies

### 3. **Debugging**
- Reproduce issue
- Isolate cause
- Test hypothesis
- Validate fix

### 4. **Optimización**
- Identify bottlenecks
- Benchmark
- Iterative improvement
- Trade-offs
