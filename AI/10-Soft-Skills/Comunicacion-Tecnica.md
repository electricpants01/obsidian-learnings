# Comunicación Técnica

## Descripción
Comunicación técnica explica conceptos complejos a audiencias variadas: stakeholders no-técnicos, management, otros engineers. Adapta mensaje según audiencia. ML context: explicar modelos, presentar resultados, documentar código, write reports. Visualizaciones efectivas, storytelling con datos, evitar jargon innecesario.

## Skills
- **Presentaciones**: Visualizaciones claras, narrativa coherente
- **Documentación**: Code comments, READMEs, arquitectura
- **Reports**: Insights accionables, no solo métricas
- **Colaboración**: Feedback constructivo, escuchar activamente
