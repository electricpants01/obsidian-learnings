# Transformers - Arquitectura

## Descripción

Transformers revolucionaron NLP con self-attention, eliminando recurrencia. Son la base de modelos como BERT, GPT, T5 y todos los LLMs modernos.

## Conceptos Clave

### 1. **Self-Attention**
- Query, Key, Value
- Scaled dot-product
- Multi-head
- Masked attention

### 2. **Arquitectura**
- Encoder
- Decoder
- Encoder-decoder
- Positional encoding

### 3. **Variantes**
- BERT (encoder)
- GPT (decoder)
- T5 (encoder-decoder)
- Vision Transformers

### 4. **Optimizaciones**
- Flash Attention
- Sparse attention
- Linear attention
- Efficient transformers
