# LangChain y LlamaIndex

## Descripción

Frameworks para construir aplicaciones con LLMs. LangChain ofrece chains y agents, mientras LlamaIndex se especializa en indexación y retrieval.

## Conceptos Clave

### 1. **LangChain**
- Chains
- Agents
- Memory
- Tools
- Callbacks

### 2. **LlamaIndex**
- Indices
- Query engines
- Retrievers
- Response synthesis

### 3. **Componentes**
- Prompts
- Output parsers
- Document loaders
- Vector stores

### 4. **Patrones**
- Sequential chains
- Router chains
- ReAct
- Tree of thought
