# AI Engineering - Guía Completa

## Fundamentos Matemáticos

1. **Álgebra Lineal** - Vectores, matrices, transformaciones
2. **Cálculo** - Derivadas, gradientes, optimización
3. **Estadística y Probabilidad** - Distribuciones, inferencia, teorema de Bayes
4. **Optimización** - Gradient descent, backpropagation

## Programación Core

5. **Python** - Lenguaje principal en AI/ML
6. **NumPy** - Computación numérica
7. **Pandas** - Manipulación de datos
8. **Git** - Control de versiones
9. **Linux/Unix** - Comandos básicos

## Machine Learning Clásico

10. **Scikit-learn** - Algoritmos ML tradicionales
11. **Algoritmos supervisados** - Regresión, clasificación
12. **Algoritmos no supervisados** - Clustering, PCA
13. **Feature Engineering** - Creación de características
14. **Validación y métricas** - Cross-validation, confusion matrix

## Deep Learning

15. **PyTorch** o **TensorFlow** - Frameworks principales
16. **Redes Neuronales** - Arquitecturas, activaciones, capas
17. **CNN** - Visión computacional
18. **RNN/LSTM/Transformers** - Secuencias y NLP
19. **Transfer Learning** - Fine-tuning de modelos
20. **Optimizadores** - Adam, SGD, etc.

## LLMs y AI Moderna

21. **Transformers** - Arquitectura fundamental
22. **Hugging Face** - Biblioteca de modelos pre-entrenados
23. **Prompt Engineering** - Diseño de prompts efectivos
24. **RAG (Retrieval Augmented Generation)** - Búsqueda + generación
25. **Vector Databases** - Pinecone, Weaviate, ChromaDB
26. **Fine-tuning LLMs** - LoRA, QLoRA, PEFT
27. **LangChain/LlamaIndex** - Frameworks para apps con LLMs
28. **OpenAI API / Anthropic API** - Integración de modelos

## MLOps y Deployment

29. **Docker** - Containerización
30. **Kubernetes** - Orquestación (opcional pero útil)
31. **MLflow** - Tracking de experimentos
32. **FastAPI/Flask** - APIs para modelos
33. **AWS/GCP/Azure** - Cloud platforms
34. **Model Serving** - TorchServe, TensorFlow Serving
35. **Monitoring** - Seguimiento de modelos en producción

## Datos y Procesamiento

36. **SQL** - Consultas de bases de datos
37. **Spark** - Procesamiento distribuido (para big data)
38. **ETL/ELT** - Pipelines de datos
39. **Data Cleaning** - Limpieza y preparación
40. **Webscraping** - BeautifulSoup, Scrapy

## Herramientas Avanzadas

41. **Jupyter Notebooks** - Experimentación
42. **Weights & Biases** - Tracking de experimentos
43. **DVC** - Versionado de datos
44. **Airflow** - Orquestación de workflows
45. **Streamlit/Gradio** - Demos y prototipos

## Especialidades Opcionales

46. **Computer Vision** - YOLO, Detectron2
47. **NLP** - spaCy, NLTK, sentiment analysis
48. **Reinforcement Learning** - Q-learning, policy gradients
49. **Time Series** - Prophet, ARIMA
50. **Generative AI** - GANs, Diffusion models

## Soft Skills Importantes

51. **Comunicación técnica** - Explicar modelos a no técnicos
52. **Pensamiento crítico** - Analizar resultados
53. **Resolución de problemas** - Debugging de modelos
54. **Aprendizaje continuo** - El campo evoluciona rápido

## Ruta de Aprendizaje Recomendada

### Fase 1: Fundamentos (2-3 meses)
- Python avanzado
- Matemáticas (álgebra lineal, cálculo)
- Pandas, NumPy, visualización

### Fase 2: ML Clásico (2-3 meses)
- Scikit-learn
- Algoritmos supervisados/no supervisados
- Feature engineering y validación

### Fase 3: Deep Learning (3-4 meses)
- PyTorch o TensorFlow
- CNNs, RNNs, Transformers
- Proyectos prácticos

### Fase 4: LLMs y AI Moderna (2-3 meses)
- Hugging Face
- Prompt engineering
- RAG, fine-tuning
- LangChain

### Fase 5: MLOps (2-3 meses)
- Docker, APIs
- Model deployment
- Monitoring y escalabilidad
