# FastAPI y Flask - APIs para ML

## Descripción

FastAPI y Flask crean APIs REST para servir modelos de ML. FastAPI es moderno y rápido, Flask es simple y maduro. Ambos son populares para deployment.

## Conceptos Clave

### 1. **FastAPI**
- Type hints
- Auto docs
- Async
- Pydantic
- Performance

### 2. **Flask**
- Lightweight
- Extensions
- Templates
- Mature ecosystem

### 3. **ML Serving**
- Model loading
- Request validation
- Error handling
- Monitoring

### 4. **Deployment**
- Docker
- Gunicorn/Uvicorn
- Load balancing
- Health checks
