# Kubernetes - Orquestación de Containers

## Descripción

Kubernetes orquesta containers a escala, gestionando deployment, scaling y operaciones de aplicaciones containerizadas. Es el estándar de facto para MLOps en producción.

## Conceptos Clave

### 1. **Arquitectura**
- Pods
- Services
- Deployments
- ConfigMaps
- Secrets

### 2. **Networking**
- ClusterIP
- NodePort
- LoadBalancer
- Ingress

### 3. **Storage**
- Volumes
- PersistentVolumes
- StorageClass

### 4. **ML Workloads**
- Model serving
- Batch inference
- Training jobs
- Auto-scaling
